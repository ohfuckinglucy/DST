```bash

docker build -t week10-app:heavy -f heavy.Dockerfile .

# Сборка легкого образа (multi-stage)
docker build -t week10-app:latest .

# Запуск
docker run -p 8278:8278 week10-app:latest
```

## Информация о слоях (Layers)
В легком образе получилось 14 слоев.
- Основные слои формируются базовым образом python:3.11-slim и Debian.
- Новые слои создают команды WORKDIR, COPY, RUN.
- Multi-stage сборка позволяет не переносить зависимости сборки (gcc, python3-dev) в финальный образ.
- Разница в размере: 1.11 GB vs 125 MB достигается за счет исключения слоя с компиляторами и кешем pip.
